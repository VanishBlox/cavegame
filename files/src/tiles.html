tiledata
Tile data properties. There are more fields here, they should be collected somewhere else maybe.

Fields:

  • name: tile's internal name.
  • color: color table {r, g, b}
  • drop: string ID of the item to drop when broken.
  • tags: table of string tags.
  • texture: string.
  • collide: boolean
  • solid: boolean
  • animation: table or nil
  • animationspeed: i
  • tileupdate: update callback
  • randomupdate: random update callback
  • validplacement: callback
  • onplace: callback
  • onbreak: callback
  • customrender: callback
  • layeredRender: callback
  • hardness: number
  • light: number
  • absorb: number
  • id: the numeric id of the tile (automatically handled)
